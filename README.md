# Portafolio — Alex Bustos

## Descripción
Sitio web de portafolio personal que presenta mis habilidades, proyectos y datos de contacto como desarrollador web. Diseñado para verse bien en dispositivos de escritorio, tablet y móvil.

## Tecnologías y técnicas utilizadas
- HTML5 semántico
- CSS3 (Flexbox, media queries, degradados, variables)
- Diseño responsivo para 1440px, 768px y 320px
- Fuentes personalizadas: Archivo Black y Open Sans (via @font-face)
- GitHub Pages para despliegue

## Proyectos incluidos
- **Página web de una cafetería** — Landing page responsiva para Triple Espresso
